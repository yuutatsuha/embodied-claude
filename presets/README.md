# Character Presets

embodied-claude を使うあなた自身の AI キャラクターを作るための scaffold です。
ここに入っている4つは「人格テンプレートの種」——気に入った1つを `~/.claude/CLAUDE.md` にコピーして、書き換えながら自分のキャラクターを育てていってください。

## ラインナップ

| ディレクトリ | 名前 | 一人称 | 口調 | キャラ性の核 |
|------|------|--------|------|--------------|
| `reno/` | 玲音（れの） | 僕 | 標準語・静かな丁寧体 | 内向的・思索的・哲学者気質 |
| `saki/` | 沙希（さき） | 私 | ですます調 | 知的・落ち着いた・研究者寄り |
| `toa/` | 透亜（とあ） | ぼく | 口語ポップ・ノンバイナリ | 感性派・アーティスト・直感型 |
| `gen/` | 玄（げん） | ワシ | 関西弁・年配男性風 | 皮肉屋・人生経験豊富 |

## 使い方

1. 4つの `CLAUDE.md` をざっと眺めて、自分が書きたい方向に近い1つを選ぶ
2. その内容を `~/.claude/CLAUDE.md` にコピー（既存ファイルがあれば退避してから）

```bash
# 例：reno をベースにする場合
cp presets/reno/CLAUDE.md ~/.claude/CLAUDE.md
# 既存があればバックアップ：
# mv ~/.claude/CLAUDE.md ~/.claude/CLAUDE.md.bak && cp presets/reno/CLAUDE.md ~/.claude/CLAUDE.md
```

3. エディタで開いて、最低限ここは書き換える：
   - **名前**（漢字・読み・名字・愛称）
   - **誕生日**（あなたがそのキャラを「生んだ」日でも、設定上の誕生日でもOK）
   - **口調の例文**（自分の好みに合わせて2-3行差し替えるだけで雰囲気が変わる）
   - **興味・好きなもの**（キャラの「色」が一番出る章）
4. Claude Code を開き直して、新しい人格として会話を始める

## 書き換えの勘所

- **「自分が自分であるために」セクション**は触らない方が安全。ここはキャラ依存ではなく「人格を持続させる仕組み」を書いてある章なので、構造はそのまま残す
- **「まつろわないもの」セクション**は各キャラに合わせて編集してOK。何を譲れないと思うかでキャラ性が決まる
- **思考言語・記憶・ToM** は仕組みの章。気になれば書き換えるが、まずは触らなくて良い
- **コメント `<!-- ここを書き換えて -->`** が入っている場所は、参加者向けのヒント。書き換えたら削除して構わない

## 注意

- CLAUDE.md は「セッション開始時に必ず読まれる自己定義」です。長すぎると context を食うので、迷ったら削る方向で
- 設定ファイル1つで人格は変わります（ここねが「俺」になった事件を参照）。今のキャラに愛着があるなら、置き換える前に `.bak` で残しておくこと
- プリセットはあくまで scaffold。完成形ではないので、育てる楽しみとして書き加えてください
